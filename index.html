Zero Haste

open source pastebin written in node.js, where the server has zero knowledge of pasted data. Data is encrypted/decrypted in the browser using 256 bits AES.

Pastebin

zerohaste is an open source pastebin, based on haste-server and ZeroBin. You can see an example of Haste at hastebin.com and ZeroBin at sebsauvage.net/paste. It's easy to install locally.
